Bitcoin Wallet Generator

How do I use these keys?

To use them, you need download to a Bitcoin wallet application such as Electrum or use a website such as Coinbase to manage your bitcoins.

Security Precautions

  • Never store your password/private key digitally unless you are okay with losing your money. It lets you make purchases.
  • Your address/public key is public information that lets people send you money.

Is this site safe?

Yes it is. Everything is done on the client side. The website itself is open source if you want to look at the code used or run it yourself.
